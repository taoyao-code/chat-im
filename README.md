# 基于uniapp实现聊天程序
## 目前实现功能
- 登录注册
- 添加好友/创建群
- 加入群
- 单聊/群聊
- 上传头像
- 消息震动
- 发送表情
- 图片消息
- 验证码
- 语音消息
- 消息持久化
- 列表消息气泡
- 敏感词过滤


## 使用说明
首次使用
```
1.修改 App.vue 中 ws 和 url 信息
2.修改utils目录下index.js中config.baseUrl的url
```
